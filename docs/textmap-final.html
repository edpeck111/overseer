WHAT'S NEW IN THIS PASS

  • Zero vertical gap. Row height crushed to 0.62em (matches JetBrains Mono's character aspect ratio) with the glyph nudged up via negative margin. Block characters now meet edge-to-edge in both axes — the map is a continuous image, not a grid.
  • Wire format optimisations applied to byte counts — newlines dropped, spatial prediction, domain arithmetic coding, multi-tile bundling. The status bar shows the optimised compressed size, ~178 B for a dense 100×36 view.
  • Hillshading. Computed from synthetic elevation data at render time (zero wire cost). Western slopes lit, eastern slopes shadowed (NW sun).
  • Contour lines. Toggleable, drawn every 50m elevation. Pure local computation.
  • Animated water. Per-cell deterministic phase delay creates a rolling shimmer across water bodies.
  • Texture variation. Forest cells use random secondary glyphs for visual variety. Moor cells too.
  • OS grid overlay. 1km national-grid lines toggleable.
  • Viewshed. Cells visible from your position rendered normal; occluded cells dimmed. Useful for radio LOS and tactical planning.
  • Distance ruler. Click two points anywhere on the map to measure. Status bar shows distance in metres/km.

EVERYTHING ABOVE IS LOCAL COMPUTATION

Hillshading, contours, water animation, texture variation, OS grid, viewshed, ruler — none of these add a single byte to the wire. They're computed from the base tile (which carries terrain class) plus locally-stored elevation data (SRTM, ~15GB for the UK on the OPi5). The base tile is the only thing that crosses the network. Everything else is rendering.

WHAT THIS GIVES OVERSEER

  • A genuinely cartographic-quality map, in pure text
  • Single-LoRa-packet tile transmission at any zoom
  • Pan operations that cost ~50 bytes each (instant on any link)
  • Rich tactical overlays (viewshed, range rings, no-go zones, routes) added at zero wire cost
  • Per-region color palettes giving each area visual identity
  • Aesthetic continuity with the rest of the BBS-flavoured Overseer UI
  • Works identically on phone WiFi, Cardputer LoRa, SSH terminal, or desktop browser
